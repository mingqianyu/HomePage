课程模块多，学习时间紧，学不会怎么办？

知识点吸收慢、落下课程怎么办？

知识点吸收慢，大部分学员是因为零基础的原因，而我们的课程设置也是针对零基础学员的。老师会根据每位学员的情况有针对性地调整授课节奏，同时也会将每次讲课的知识点录成视频，发给学员去复习，遇到问题可以随时找老师一对一指导解决。

我喜欢编程，不知道是否合适学习python？

爱好兴趣是成功最大的法宝，有兴趣就可以全身心地投入去学习，也更容易保持专注，加上有我们的专业老师在指导，学习起来自然事半功倍。现实中有很多零基础小白都能够在我们的指导下，顺利成为Python开发工程师，不但就业普遍高薪，而且入职后的晋升发展也反映良好。

知识点吸收慢、落下课程怎么办？

作为资深程序员本身具备的一些技能是完全可以跨行业应用的，比如软件技法。Python学精较难，由浅入深、循序渐进，才能夯实基础，注意结合潮流开发技术，尤其是前后端交互、大数据、AI智能等，才更容易拿高薪。同时，互联网更新快，时间与开发成本也是Python开发工程师要注意的。

如何保证学习效果质量的？

蓝鸥Python课程设计严格遵循系统化、体系化设计，完全与潮流技术同步。课程涵盖爬虫技术、大数据分析技术、人工智能等，学完即能成为掌握Python核心技术的全栈开发工程师。全面覆盖从小程序开发到名企项目全案的项目：一线最新应用技术，紧贴企业用人需求培养。

是否可以免费重修？

学生如果因为中途落课或学习效果不扎实等原因，可以向教务老师申请降级重修。本期学不会或学不扎实的，可以在下期免费重学，直到学会为止！
